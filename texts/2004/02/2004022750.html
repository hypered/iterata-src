Titre
8 SEPTEMBRE 2004. - Arrêté ministériel portant prolongation de l'agrément de certaines banques de tissus.

Source : SANTE PUBLIQUE, SECURITE DE LA CHAINE ALIMENTAIRE ET ENVIRONNEMENT
Publication : 06-10-2004 numéro :   2004022750 page : 70387       PDF :   version originale    
Dossier numéro : 2004-09-08/31
Entrée en vigueur / Effet : 01-07-2004

Table des matières Texte Début
Art. 1-3

Texte Table des matières Début
Article 1. Les organismes suivants voient leur agrément prolongé jusqu'au 31 décembre 2006, dans les conditions suivantes :
  1° AZ Sint-Augustinus, A.S.B.L., Oosterveldlaan 24, 2610 Wilrijk (Antwerpen), pour :
  a) Les allogreffes de têtes fémorales;
  b) Les allogreffes tympano-ossiculaires;
  2° Algemeen Ziekenhuis Sint-Dimpna, J.B. Stessensstraat 2, 2440 Geel, pour les allogreffes de têtes fémorales;
  3° Algemeen Ziekenhuis Sint-Jan, Ruddershove 10, 8000 Brugge, pour :
  a) Les allogreffes du système locomoteur;
  b) Les allogreffes de tissus ophtalmiques;
  4° Ziekenhuis Netwerk Antwerpen - Campus Stuivenberg, Lange Beeldekensstraat 267, 2060 Antwerpen, pour les allogreffes de peau;
  5° Academisch Ziekenhuis VUB, Laarbeeklaan 101, 1090 Jette - Bruxelles, pour :
  a) Les allogreffes du système locomoteur;
  b) Les allogreffes de cellules bèta-pancréatiques;
  6° Clinique Universitaire Saint-Luc (UCL), avenue E. Mounier 53, 1200 Bruxelles, pour les allogreffes de l'appareil locomoteur;
  7° Monica ziekenhuis, campus Eeuwfeestkliniek, Harmoniestraat 68, 2018 Antwerpen, pour les allogreffes de têtes fémorales;
  8° European Homograft Bank, rue Bruyn 1, 1120 N.O.H. Bruxelles, pour :
  a) Les allogreffes valvulaires;
  b) Les allogreffes vasculaires;
  9° Hôpital militaire, Centre des brûlures, rue Bruyn 1, 1120 N.H.O. Bruxelles, pour les allogreffes de peau de donneur;
  10° Intercommunale Famenne Ardenne Condroz (IFAC), rue du Vivier 21, 6900 Aye-Marche, pour les allogreffes de têtes fémorales;
  11° Universitair Ziekenhuis Gent, De Pintelaan 185, 9000 Gent, pour :
  a) Les allogreffes de tissus ophtalmiques;
  b) Les allogreffes vasculaires;
  c) Les allogreffes tympano-ossiculaires;
  d) Les allogreffes du système locomoteur;
  e) Les kératinocytes;
  12° Universitair Ziekenhuis Antwerpen, Wilrijkstraat 10, 2650 Edegem, pour
  a) Les allogreffes de tissus ophtalmiques;
  b) Les allogreffes tympano-ossiculaires;
  c) Les allogreffes du système locomoteur;
  13° Universitaire Ziekenhuizen KUL, Herestraat 49, 3000 Leuven, pour :
  a) Les allogreffes de tissus ophtalmiques;
  b) Les allogreffes de peau de donneurs;
  c) Les allogreffes tympano-ossiculaires;
  d) Les allogreffes du système locomoteur;
  e) Les kératinocytes;
  14° Algemeen Ziekenhuis Sint-Elisabeth - Herentals, Nederrij 133, 2200 Herentals, pour les allogreffes de têtes fémorales;
  15° Sint-Vincentiusziekenhuis V.Z.W., Campus Sint-Vincentius, Sint-Vincentiusstraat 20, 2018 Antwerpen, pour les allogreffes de têtes fémorales;
  16° Virga Jesseziekenhuis, Stadsomvaart 11, 3500 Hasselt, pour les allogreffes de têtes fémorales;
  17°) CHIREC, site clinique Edith Cavell, rue Edith Cavel 32, 1180 Bruxelles, pour les allogreffes de têtes fémorales;
  18° Heilige Hartziekenhuis, Wilgenstraat 2, 8800 Roeselare, pour les allogreffes de têtes fémorales;
  19° Algemeen ziekenhuis Zusters van Barmhartigheid, Glorieuxlaan 55, 9600 Ronse, pour les allogreffes de têtes fémorales.

  Art. 2. L'organisme suivant voit son agrément prolongé jusqu'au 31 décembre 2004 dans les conditions suivantes :
  Hôpital militaire, Centre des brûlures, rue Bruyn 1, 1120 N.H.O. Bruxelles, pour :
  Les allogreffes de kératinocytes.

  Art. 3. Le présent arrêté produit ses effets au 1er juillet 2004.
  Bruxelles, le 8 septembre 2004.
  R. DEMOTTE.

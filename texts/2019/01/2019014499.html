Titre
29 AOUT 2019. - Arrêté royal modifiant l'annexe IIIbis de l'AR/CIR 92 en matière de la dispense de versement de précompte professionnel visée à l'article 2751 du Code des impôts sur les revenus 1992

Source : FINANCES
Publication : 16-09-2019 numéro :   2019014499 page : 86447       PDF :   version originale    version consolidée
Dossier numéro : 2019-08-29/07
Entrée en vigueur / Effet : 01-01-2019

Art. 1-3

Texte Table des matières Début
Article 1er. A l'annexe IIIbis de l'AR/CIR 92, insérée par l'arrêté royal du 22 août 2006, remplacée par l'arrêté royal du 31 juillet 2009 et modifiée par les arrêtés royaux des 21 février 2014, 28 avril 2015, 23 août 2015, 19 juillet 2018 et 13 avril 2019, les modifications suivantes sont apportées :
  1° le code "44 heures supplémentaires (Art. 2751, alinéa 4, 2ème tiret, CIR 92) est remplacé par le code "44 heures supplémentaires (art. 2751, alinéa 4, deuxième tiret, et alinéa 7, CIR 92)" ;
  2° le code "45 heures supplémentaires (Art. 2751, alinéa 4, 1er tiret, CIR 92)" est remplacé par le code "45 heures supplémentaires (art. 2751, alinéa 4, premier tiret, et alinéa 7, CIR 92)" ;
  3° le code "51 heures supplémentaires (art. 2751, alinéa 4, 2e tiret, et art. 2751, alinéa 7, CIR 92)" est remplacé par le code "51 heures supplémentaires (art. 2751, alinéa 4, deuxième tiret, et alinéa 8, CIR 92)" ;
  4° le code "52 heures supplémentaires (art. 2751, alinéa 4, 1er tiret, et art. 2751, alinéa 7, CIR 92)" est remplacé par le code "52 heures supplémentaires (art. 2751, alinéa 4, premier tiret, et alinéa 8, CIR 92)" ;
  5° le code "55 Horeca (Art. 2751, alinéa 4, deuxième tiret, et alinéa 8, CIR 92)" est remplacé par le code "55 Horeca (art. 2751, alinéa 4, deuxième tiret, et alinéa 9, CIR 92)".

  Art. 2. Le présent arrêté produit ses effets aux rémunérations payées ou attribuées à partir du 1er janvier 2019.

  Art. 3. Le ministre qui a les Finances dans ses attributions est chargé de l'exécution du présent arrêté.

   Considérant qu'il s'agit d'un simple arrêté exécutant une législation existante, et que le présent arrêté n'a en soi aucune nouvelle conséquence budgétaire ;
   Vu les lois sur le Conseil d'Etat, coordonnées le 12 janvier 1973, l'article 3, § 1er ;
   Vu l'urgence ;
   Considérant :
   - que la loi du 23 mars 2019 modifiant le Code des impôts sur les revenus 1992 en ce qui concerne les dispositions fiscales relatives au deal pour l'emploi a modifié l'article 2751 du Code des impôts sur les revenus 1992 ;
   - qui suite à cette modification, le nombre d'heures de travail supplémentaire pour lesquelles une dispense de versement de précompte professionnel peut être octroyée, a été porté de 130 à 180 heures par an pour les années 2019 et 2020 ;
   - que les débiteurs du précompte professionnel doivent être informés de quels codes ils doivent utiliser pour cette mesure dans la deuxième déclaration au précompte professionnel qu'ils doivent introduire en application de l'article 952, AR/CIR 92 ;

Rapport au Roi Texte Table des matières Début
   RAPPORT AU ROI
   Sire,
   La loi du 23 mars 2019 modifiant le Code des impôts sur les revenus 1992 en ce qui concerne les dispositions fiscales relatives au deal pour l'emploi a porté le nombre d'heures de travail supplémentaire pour lesquelles une dispense de versement de précompte peut être accordée de 130 à 180 heures pour les années 2019 et 2020.
   Pour l'application de la dispense de versement de précompte professionnel, les débiteurs du précompte doivent introduire une deuxième déclaration au précompte professionnel. Dans cette deuxième déclaration, ils doivent indiquer de quelle mesure ils veulent bénéficier en utilisant les codes repris dans l'annexe IIIbis de l'AR/CIR 92. Actuellement, il y a cinq codes pour la dispense de versement de précompte pour le travail supplémentaire :
   - le code 44 pour les premières 130 heures supplémentaires avec un sursalaire de 50 ou 100 p.c. ;
   - le code 45 pour les premières 130 heures supplémentaires avec un sursalaire de 20 p.c. ;
   - le code 51 pour 50 heures supplémentaires additionnelles lorsque des travaux immobiliers sont exécutés sur des chantiers avec un système électronique d'enregistrement des présences et qu'un sursalaire de 50 ou 100 p.c. s'applique ;
   - le code 52 pour 50 heures supplémentaires additionnelles lorsque des travaux immobiliers sont exécutés sur des chantiers avec un système électronique d'enregistrement des présences et qu'un sursalaire de 20 p.c. s'applique ;
   - le code 55 pour 230 heures supplémentaires additionnelles dans l'horeca ;
   Pour les années 2019 et 2020 le nombre d'heures supplémentaires pour lesquelles une dispense de versement de précompte professionnel peut être accordée, est augmenté de 130 à 180 heures pour les travailleurs dans tous les secteurs. L'impact de cette augmentation sur le marché du travail sera évalué par le gouvernement et, le cas échéant, il peut être décidé de rendre permanente cette mesure temporaire.
   Pour les années 2019 et 2020, la dispense de versement de précompte professionnel pour les 50 heures supplémentaires additionnelles pour les travailleurs pour qui les augmentations existantes ne s'appliquent pas, devra toujours être postulée en utilisant les codes "ordinaires", c.-à-d. les codes 44 et 45. Pour les 50 ou 230 heures supplémentaires additionnelles qui tombent dans le champ d'application des augmentations existantes (l'augmentation pour des travaux immobiliers exécutés sur des chantiers avec un système électronique d'enregistrement des présences et l'augmentation pour l'horeca), la dispense de versement de précompte professionnel devra comme avant être postulée en utilisant, selon le cas, les codes 51, 52 et 55. Dans le cadre de l'évaluation de l'impact de la mesure, l'on pourra ainsi mieux déterminer quels débiteurs du précompte professionnel font appel à l'augmentation temporaire. Les références aux parties pertinentes de l'article 2751 du Code des impôts sur les revenus 1992 dans le libellé des codes 44, 45, 51, 52 et 55 dans l'annexe IIIbis, AR/CIR 92 sont adaptées. Il va de soi que l'administration n'exigera pas pour les années 2019 et 2020 que les employeurs qui utilisent les codes 51 et 52 apportent la preuve qu'il est satisfait aux conditions spécifiques pour l'augmentation pour les travaux immobiliers pour pouvoir bénéficier de la dispense pour 50 heures supplémentaires additionnelles.
   Dans la lignée de l'entrée en vigueur de l'article 7 de la loi du 23 mars 2019 modifiant le Code des impôts sur les revenus 1992 en ce qui concerne les dispositions fiscales relatives au deal pour l'emploi, ces modifications produisent leurs effets aux rémunérations payées ou attribuées à partir du 1er janvier 2019.
   Telle est, Sire, la portée de l'arrêté qui Vous est soumis.

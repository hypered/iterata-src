Titre
18 JANVIER 2019. - Arrêté du Gouvernement flamand fixant la liste de communes, visée à [l'article 5.78, alinéa 1er, du Code flamand du Logement de 2021], dans le cadre de l'évaluation d'avancement de 2018 <Intitulé modifié par AGF 2020-07-17/73, art. 86, 002; En vigueur : 01-01-2021>
(NOTE : Consultation des versions antérieures à partir du 11-03-2019 et mise à jour au 17-11-2020)

Art. 1-2

Texte Table des matières Début
Article 1er.En application de [1 l'article 5.78, alinéa 2, du Code flamand du Logement de 2021]1, la liste des communes suivantes est fixée :
  1° Anvers ;
  2° Baerle-Duc ;
  3° Boom ;
  4° Bornem ;
  5° Termonde ;
  6° Dilsen-Stokkem ;
  7° Duffel ;
  8° Eeklo ;
  9° Genk ;
  10° Gand ;
  11° Hamme ;
  12° Herenthout ;
  13° Ypres ;
  14° Courtrai ;
  15° Kuurne ;
  16° Lokeren ;
  17° Maaseik ;
  18° Maasmechelen ;
  19° Malines ;
  20° Menin ;
  21° Messines ;
  22° Nieuwpoort;
  23° Olen ;
  24° Schelle ;
  25° Sint-Amands ;
  26° Saint-Trond ;
  27° Espierres-Helchin ;
  28° Tamise ;
  29° Furnes ;
  30° Vilvorde ;
  31° Wachtebeke ;
  32° Wervik ;
  33° Willebroek ;
  34° Zelzate ;
  35° Zwevegem ;
  36° Zwijndrecht.
  ----------
  (1)<AGF 2020-07-17/73, art. 87, 002; En vigueur : 01-01-2021>

  Art. 2. Le Ministre flamand ayant le logement dans ses attributions est chargé de l'exécution du présent arrêté.

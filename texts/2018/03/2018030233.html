Titre
22 DECEMBRE 2017. - Arrêté du Gouvernement flamand modifiant diverses dispositions de l'arrêté du Gouvernement flamand du 17 février 2017 portant octroi d'allocations de primes à l'embauche de demandeurs d'emploi de longue durée

Source : AUTORITE FLAMANDE
Publication : 12-02-2018 numéro :   2018030233 page : 10161       PDF :   version originale    
Dossier numéro : 2017-12-22/52
Entrée en vigueur / Effet : 01-01-2018

Ce texte modifie le texte suivant :2017030126       

Table des matières Texte Début
Art. 1-10

Texte Table des matières Début
Article 1er. L'article 1er de l'arrêté du Gouvernement flamand du 17 février 2017 portant octroi d'allocations de primes à l'embauche de demandeurs d'emploi de longue durée, est complété par un point 6°, rédigé comme suit :
  " 6° Département : le Département de l'Emploi et de l'Economie sociale, visé à l'article 25, § 1er, de l'arrêté du Gouvernement flamand du 3 juin 2005 relatif à l'organisation de l'Administration flamande. ".

  Art. 2. Dans l'article 2, § 1er, alinéa 1er, phrase introductive, du même arrêté, les mots " le VDAB " sont remplacés par les mots " le Département ".

  Art. 3. A l'article 6 du même arrêté, les modifications suivantes sont apportées :
  1° dans le paragraphe 1er, alinéa 1er, les mots " le VDAB " sont remplacés par les mots " le Département " et les mots " Le VDAB " sont remplacés par les mots " Le Département " ;
  2° dans le paragraphe 1er, alinéa 3, les mots " le VDAB " sont remplacés par les mots " le Département " ;
  3° dans le paragraphe 1er, alinéa 4, les mots " Le VDAB " sont remplacés par les mots " Le Département " ;
  4° dans le paragraphe 1er, alinéa 5, les mots " Le VDAB " sont remplacés par les mots " Le Département " ;
  5° dans le paragraphe 2, alinéas 1er et 3, les mots " le VDAB " sont remplacés par les mots " le Département ".

  Art. 4. Dans l'article 7 du même arrêté, les mots " Le VDAB " sont chaque fois remplacés par les mots " Le Département ".

  Art. 5. Dans l'article 8, alinéas 1er et 3, du même arrêté, les mots " le VDAB " sont remplacés par les mots " le Département ".

  Art. 6. A l'article 9 du même arrêté les modifications suivantes sont apportées :
  1° dans le paragraphe 1er, alinéa 1er, les mots " Le VDAB " sont remplacés par les mots " Le Département " et les mots " le VDAB " sont remplacés par les mots " le Département " ;
  2° dans le paragraphe 1er, alinéa 2, les mots " le VDAB " sont remplacés par les mots " le Département " ;
  3° dans le paragraphe 2 et le paragraphe 3, alinéa 1er, les mots " le VDAB " sont remplacés par les mots " le Département " ;
  4° dans le paragraphe 3, alinéa 1er, les mots " le VDAB " sont remplacés par les mots " le Département " ;
  5° dans le paragraphe 4, les mots " Le VDAB " sont remplacés par les mots " Le Département ".

  Art. 7. Dans l'article 10 et l'article 11, alinéa 1er, du même arrêté, les mots " le VDAB " sont remplacés par les mots " le Département ".

  Art. 8. Dans l'article 12 du même arrêté, les mots " Le VDAB " sont remplacés par les mots " Le Département ".

  Art. 9. Le présent arrêté entre en vigueur le 1er janvier 2018.

  Art. 10. Le ministre flamand qui a la politique de l'emploi dans ses attributions est chargé de l'exécution du présent arrêté.
